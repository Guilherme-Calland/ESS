git checkout master --> apontar para o master 
git merge nome_branch --> juntar as novidades do novo_branch e do master
                          dentro do master(ponteiro atual)

git pull --> o contrário, faz o merge com o que vc ta mexendo 
com o do repo

nao quer lhe permitir pazer push no projeto)
